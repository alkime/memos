Welcome to Memos, a modern platform designed to transform your voice recordings into beautifully formatted blog posts.

## What is Memos?

Memos is a static site generator platform built with Hugo and served by a lightweight Go web server. It's designed from the ground up to make blogging as simple as speaking your thoughts.

## Key Features

- **Voice-First**: Designed for converting voice recordings into written content
- **Fast & Lightweight**: Powered by Hugo static site generation
- **Modern Stack**: Built with Go, Hugo, and deployed on Fly.io
- **Clean Design**: Using the beautiful PaperMod theme

## Getting Started

This is just the beginning. We're currently in Phase I of development, establishing the core infrastructure that will power the voice-to-blog workflow.

Stay tuned for more updates as we build out the platform!

## Technical Stack

- **Hugo**: Static site generator
- **Go + Gin**: Web server
- **PaperMod**: Clean, minimal Hugo theme
- **Fly.io**: Deployment platform
- **Docker**: Containerization

We're excited to share this journey with you!
